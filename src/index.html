Resources

GET /ingredients

Returns a list of all ingredients.

curl ingredients

POST /ingredients

Creates a new ingredient.

curl ingredients -X POST -d '{"name":"kale"}' --header 'content-type: application/json'

GET /ingredients/:ingredientId

Returns a single ingredient.

curl ingredients/1

PUT /ingredients/:ingredientId

Updates an existing ingredient.

curl ingredients/1 -X PUT -d '{"name":"kale"}' --header 'content-type: application/json'

DELETE /ingredients/:ingredientId

Deletes an existing ingredient.

curl ingredients/1 -X DELETE
